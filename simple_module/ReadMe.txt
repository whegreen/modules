This sample implements a Math module, simple_math.dll,
that exposes the following functions:
(1) Synchronous Invert
(2) Asynchronous Invert
(3) Invert with power management

The sample loads the Math module and calculates
the inversion of number 0.3 using the above functions.

Run this sample under command line:

simple_module.exe

There is no command line argument.
